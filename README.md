# learning-git

DESCRIPTION 

The Student Management System is a console-based Java application developed to manage student records efficiently.
It allows users to add, view, search, and delete student details, reducing manual effort and errors.
This project is designed for beginners to understand Object-Oriented Programming (OOP) concepts in Java.

 OBJECTIVES 
To store student information digitally
To perform basic CRUD operations
To learn Java fundamentals and OOP concepts
To build a beginner-friendly mini project
FEATURES
Add new student records
View all students
Search student by roll number
Delete student record
Menu-driven console interface
 Technologies Used
Programming Language: Java
IDE: VS Code / Eclipse / IntelliJ
Concepts:
Classes & Objects
ArrayList
Scanner
Switch Case
Loops
Methods
